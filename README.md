## 📖 Sobre o Projeto

O **FirstStep** é uma plataforma completa de onboarding desenvolvida para os novos colaboradores da **Vivo (Telefônica Brasil)**. 

A solução nasceu com o objetivo de centralizar a jornada de integração, conectando os novos colaboradores (*onboardees*) com mentores experientes (*buddies*). Além disso, conta com um poderoso painel administrativo para o setor de RH gerenciar cursos, acessos a sistemas e designações de forma ágil. 

O projeto foi desenvolvido totalmente do zero, possuindo uma interface fluida, **totalmente responsiva** e com um forte foco em **acessibilidade** em qualquer dispositivo (mobile, tablet ou desktop).

🔗 **Acesse o projeto online:** [FirstStep - Vivo Challenge](https://luanrodrigues15.github.io/firststep-vivo-challenge/)

A versão original e arquitetada para produção contava com integração completa a um banco de dados real (back-end).

## ✨ Principais Funcionalidades

O sistema é dividido em três perfis principais de acesso:

### 👤 1. Perfil Onboardee (Novo Colaborador)
- Visualização da jornada de integração passo a passo.
- Acesso à central de documentos e status de envio.
- Catálogo de cursos obrigatórios e trilhas de aprendizado.
- Solicitação e acompanhamento de acessos a sistemas internos.
- Contato e acompanhamento com o *Buddy* designado.

### 🤝 2. Perfil Buddy (Mentor)
- Dashboard para acompanhamento do progresso dos seus *onboardees*.
- Visualização de métricas de engajamento dos novatos na integração.

### ⚙️ 3. Perfil Administrativo (RH)
- Gestão de usuários (cadastro, edição e designação de perfis).
- Atribuição de *Buddies* para os novos colaboradores.
- Gerenciamento de acessos a plataformas e sistemas da empresa.
- Administração de cursos e acompanhamento de métricas.

## 🛠️ Tecnologias e Bibliotecas Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias e ecossistemas modernos:

- **[React 18](https://react.dev/)** + **[Vite](https://vitejs.dev/)**: Core da aplicação e bundler de alta performance.
- **[TypeScript](https://www.typescriptlang.org/)**: Tipagem estática para maior segurança e escalabilidade.
- **[Tailwind CSS](https://tailwindcss.com/)**: Estilização utilitária, responsiva e ágil.
- **[Shadcn UI](https://ui.shadcn.com/)** / **[Radix UI](https://www.radix-ui.com/)**: Biblioteca de componentes acessíveis e customizáveis.
- **[React Router DOM](https://reactrouter.com/)**: Gerenciamento de rotas e navegação da SPA.
- **[React Hook Form](https://react-hook-form.com/)** + **[Zod](https://zod.dev/)**: Gerenciamento e validação de formulários complexos.
- **[Recharts](https://recharts.org/)**: Renderização de gráficos para os dashboards.
- **[Lucide React](https://lucide.dev/)**: Pacote de ícones.
